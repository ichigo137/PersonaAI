# PersonaAI

An AI assistant built with Kotlin and Jetpack Compose.

## Features

- Modern Material 3 UI
- Chat interface
- AI abstraction layer
- Room chat history
- Retrofit networking
- Hilt Dependency Injection
- MVVM + Clean Architecture

## Architecture

presentation
↓
domain
↓
repository
↓
provider
↓
network
